# rust-async-spider

Just a clean boilerplate for crawling website & scrapping data with: 

- async-std
- surf
- scraper
- colored

## How to use?

``` bash
git clone git@github.com:zzy/rust-async-spider.git
cd rust-async-spider
cargo build
cargo run
```
