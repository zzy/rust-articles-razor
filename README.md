# rust-async-crawler

Just a clean boilerplate for crawling website with: 

- async-std
- surf
- scraper
- colored

## How to use?

``` bash
git clone git@github.com:zzy/rust-async-crawler.git
cd rust-async-crawler
cargo build
cargo run
```
